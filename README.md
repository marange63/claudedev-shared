\# claudedev-shared



Shared utilities used by multiple PyCharm projects inside ClaudeDev.



\## Purpose



This repository contains code that is reused across multiple local Python projects.



\## Development install



From the repository root, inside the shared library's conda environment:



&nbsp;   python -m pip install -e ".\[dev]"



\## Consumer install



From any consumer project's conda environment:



&nbsp;   python -m pip install -e "C:\\Users\\YOUR\_USER\\ClaudeDev\\libs\\claudedev-shared"
